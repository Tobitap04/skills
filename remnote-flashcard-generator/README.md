# RemNote Flashcard Generator

Generate optimal, evidence-based flashcards in RemNote format from lecture PDFs and study notes.

## Features

- **Multiple input scenarios:** PDF-only, notes-only, or combined PDF+notes processing
- **Cognitive science principles:** Minimum information principle, active recall, spaced repetition
- **Intelligent formatting:** Bold keywords, inline code, italic, LaTeX equations
- **Smart card selection:** Automatically chooses best RemNote card type for each question
- **Content optimization:** Definition-first ordering, overview-before-details for multiple methods
- **Redundancy handling:** Consolidates scattered questions, filters excessive content
- **Mathematical notation:** Explains complex symbols inline within cards

## Usage

Invoke the skill and provide:
- A lecture PDF file (`.pdf`)
- Study notes (text/markdown)
- Or both for comprehensive refinement

The skill outputs flashcards ready to paste directly into RemNote.
